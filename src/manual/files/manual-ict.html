Alles om je startpagina te beheren

Om je startpagina te beheren, moet je eerst Visual Studio Code installeren. Dat is een HTML-editor van Microsoft.

Als je deze hebt gedownload, moet je in Visual Studio Code jouw eigen startpagina openen.

1. Links beheren

Open je startpagina en scroll naar onder. Je ziet nu als het goed is ongeveer dit:


1. Link openen op nieuw tabblad

Met dit element kunt u beheren of de bijbehorende link wordt geopend op een nieuw tabblad. Als u dat wel wilt, moet u dit laten staan. Als u dit niet wilt, moet u target="_blank" laten staan.

2. De link

Hiermee kunt u beheren naar welke website u de gebruikers wilt doorsturen. Vervang [link] door uw eigen link.

3. De afbeelding

Hiermee beheert u de link van de afbeelding. Op PostImages kunt u een afbeelding uploaden. Bij de volgende stap moet u de directe link kopiÃ«ren en op de plek van [link afbeelding] plakken.

4. De naam

Als u naam link wijzigt, wijzigt u de tekst die te zien is op de startpagina. Dit is de naam van de link.
